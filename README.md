# 📋 Diário de Treinos

Sistema Full Stack desenvolvido como projeto final da disciplina de Desenvolvimento Web na plataforma OxeTech.

## Funcionalidades

- Cadastro de usuários
- Login com autenticação JWT
- Cadastro de exercícios
- Listagem de exercícios
- Edição de exercícios
- Exclusão de exercícios

## Tecnologias

### Frontend

- HTML5
- CSS3
- JavaScript

### Backend

- Node.js
- Express

### Banco de Dados

- PostgreSQL

### Segurança

- JWT
- bcrypt

## Estrutura do Projeto

frontend/
src/
database.sql

## Como executar

1. npm install
2. Criar o banco diario_treinos
3. Configurar o .env
4. npm run db:init
5. npm run dev
